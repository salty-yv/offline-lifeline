# 迷路

## 先做这 3 步

1. 停下来，避免继续盲目走远。
2. 找相对安全、避风、不会滑坠的位置。
3. 降低屏幕亮度，准备一条包含位置线索、受伤情况、电量的求救信息。

## 接下来几分钟

- 回忆最后一个确定地点、岔路、标识、河流或建筑。
- 用衣物、石块或屏幕 SOS 做明显标记。
- 天色变暗或受伤时，优先保暖和等待救援。

## 不要做

- 不要为了"试试看"继续走未知路线。
- 不要频繁开关高耗电功能。
- 不要在陡坡、河边、崖边停留。
