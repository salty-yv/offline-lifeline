# 中毒

## 先做这 3 步

1. **立即呼救并联系专业机构：**保持冷静，立即拨打当地紧急电话或联系中毒控制中心（美国为 1‑800‑222‑1222）寻求指导，不要等待症状出现。
2. **停止暴露并保留证据：**若是误食毒物，立即让患者吐出口中残留物，保留所食用的物品或包装供医生分析；若接触到皮肤或眼睛，立即用大量清水冲洗，脱去受污染衣物。
3. **不要自行催吐：**不要根据包装说明或民间偏方自行催吐或服用解毒药，除非毒物控制中心或医生明确要求。

## 接下来几分钟

- 在电话指导下，记录患者年龄、体重、食用或接触毒物的名称、数量和时间，并将相关包装或样品一起送往医疗机构。
- 若患者清醒且医生建议，可少量饮用清水以稀释部分毒物；不要自行给予牛奶、酒或其他解毒剂。
- 密切观察是否出现喉咙灼痛、呼吸困难、瞳孔异常、嗜睡或抽搐等症状，如有任何严重反应立即拨打急救电话；必要时按照 [休克](/medical/shock.md) 指南处理。

## 不要做

- 不要等待症状出现再求助；任何可疑中毒都应立即联系专业机构。
- 不要让患者自行催吐或喝大量水或酒，催吐可能导致毒物进入气管，饮酒会加重中毒。
- 不要盲目服用药物中和毒物，不要跟随包装上的旧式催吐建议；如果患者无意识、抽搐或呼吸困难，请勿喂食或喂水，立即求救。
