# 误食

## 先做这 3 步

1. 立刻停止继续食用。
2. 保留剩余食物、包装或照片。
3. 出现呕吐、腹痛、意识异常或呼吸困难时尽快求救。

## 接下来几分钟

- 记录食用时间、数量和症状。
- 清醒时可少量饮水，等待专业指导。

## 不要做

- 不要判断未知蘑菇、野果、植物一定可食用。
- 不要自行催吐，除非专业人员明确要求。
