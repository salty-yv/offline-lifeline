# 热射病

## 先做这 3 步

1. **立即呼救并转移：**热射病是一种危及生命的急症。一旦发现患者体温极高、意识不清或抽搐，应立即拨打紧急电话求救，并将患者移至阴凉或空调环境。
2. **迅速降温：**脱去多余衣物，用冷水淋浴、浸泡或用湿毛巾擦拭身体，如条件允许可将患者全身浸入冷水浴中。没有盆浴时，可用凉水喷洒并持续扇风，同时在颈部、腋下和腹股沟放置冰袋以加快散热。
3. **监测生命体征并补充液体：**如患者仍保持清醒，让其小口饮用冷水或含电解质饮料；若出现昏迷或抽搐，应随时检查呼吸与脉搏，并准备进行心肺复苏术。在等待救护车期间，密切观察并准备按照 [休克](/medical/shock.md) 指南处理。

## 接下来几分钟

- **持续降温：**在专业人员到来前持续用冷水、湿毛巾或冰袋降温，直至体温降至安全范围（约 38.5 °C）。注意不要过度降温导致寒战。
- **保持仰卧与通气：**让患者仰卧并稍微抬高双腿，松开紧身衣物，保持空气流通；如出现呕吐或痉挛，可将其置于稳定侧卧位，保证气道通畅，并随时准备实施心肺复苏。
- **防止复发并记录信息：**记录发病时间、所采取的降温措施和患者的反应，随身携带至医院；在高温环境中活动时注意防暑降温，参见 [中暑](/medical/heatstroke.md) 和 [脱水](/medical/dehydration.md) 指南。

## 不要做

- 不要让患者自行站立或行走，防止跌倒和血压骤降。
- 不要给患者饮用含酒精或咖啡因的饮料，也不要一次大量灌水，尤其在意识不清时。
- 不要使用过冰冷的水或冰袋直接覆盖全身，以免血管过度收缩反而降低散热效率；不要等待症状自行缓解而延误救治。
