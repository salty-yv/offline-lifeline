# 扭伤与疑似骨折

## 先做这 3 步

1. 立刻停止负重和继续行走。
2. 用衣物、木棍或背包带固定受伤部位。
3. 如果出现变形、麻木、剧痛或无法移动，按疑似骨折处理。

## 接下来几分钟

- 保持患处稳定，减少移动。
- 有条件时冷敷，但不要让皮肤冻伤。
- 结合迷路、夜间、低温风险决定是否原地等待。

## 不要做

- 不要强行复位。
- 不要忍痛继续赶路。
