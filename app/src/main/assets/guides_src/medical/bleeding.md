# 出血

## 先做这 3 步

1. **确保安全并呼救：** 快速评估周围环境让自己和伤者脱离危险，大声呼叫他人或拨打当地紧急电话求助。
2. **加压止血：** 用干净的敷料、衣物或绷带直接压住出血点，同时尽量让伤处高于心脏，持续用力压迫以减缓出血。当敷料被血液浸透时不要揭开查看，而是在原位上再覆盖一层并继续压迫。
3. **必要时使用止血带：** 如果四肢出现严重出血且直接加压无法控制，可在靠近伤口的上方绑扎止血带，只在生命危险情况下使用，并记录绑扎时间以方便医护人员。

## 接下来几分钟

- 持续按压伤口，定期检查绑扎部位远端的肤色和温度，防止包扎过紧导致血液循环受阻。
- 让伤者平卧并保暖，关注呼吸、脉搏和意识状况；如果出现皮肤苍白、出冷汗或脉搏微弱等休克征象，请参阅 [休克](/medical/shock.md) 指南处理。
- 若出血持续不止或出现喷射状出血，立即增加压迫并尽快呼叫救援，同时记录受伤时间、出血部位和失血量供医护人员参考。

## 不要做

- 不要用泥土、草药或其他不洁材料塞入伤口，也不要自行切开或放血以免感染或加重损伤。
- 不要频繁掀开敷料观察是否止血；不要包扎过紧导致肢端发紫麻木。
- 除非专业人员指导或生命危险，不要在非肢体部位使用止血带，也不要擅自移除深嵌异物。
