# 过敏反应

## 先做这 3 步

1. **评估症状并立即用药：**如果仅出现瘙痒、荨麻疹或轻度局部肿胀，可在有条件的情况下立即服用口服抗组胺药；如出现呼吸困难、喉咙紧缩、面部或舌头肿胀、声音嘶哑或意识混乱等严重症状，应立刻使用肾上腺素自动注射器（俗称 “EpiPen”），并拨打紧急电话。
2. **让患者平卧并保持安静：**在等待救援时，让患者平躺并抬高双腿，除非其呼吸困难或呕吐；按压自动注射器于大腿外侧，持续 3 秒完成注射。保持安静可减缓过敏反应。
3. **准备重复注射并求救：**如果 5 分钟后症状未缓解或复发，可使用第二支肾上腺素注射器并再次拨打急救电话，告诉救援人员已使用肾上腺素。立即联系患者的家属或急救联系人，并准备按照 [休克](/medical/shock.md) 指南处理。

## 接下来几分钟

- **持续观察并保持平卧：**切勿让患者站立或走动，以免血压骤降引发休克。监测呼吸、脉搏和意识水平，并随时准备实施心肺复苏；如果患者出现呕吐或不能正常呼吸，可将其置于稳定侧卧位并清理口腔，确保气道通畅。
- **应对轻度过敏：**如果反应较轻，抗组胺药可能缓解症状，但仍需观察数小时以防进展。可用冷敷或涂抹止痒药膏缓解局部症状；如果症状加重或扩散，应立即寻求医疗帮助。
- **防止复发并随身备用：**对于已知有严重过敏史的人士，应随身携带两支肾上腺素注射器。过敏反应缓解后，仍需前往医院观察 4 小时，以便处理潜在的迟发反应。

## 不要做

- 不要只依赖抗组胺药来处理严重的过敏反应；肾上腺素是唯一一线治疗。
- 不要给呼吸困难或意识不清的患者口服药物或水，以防误吸；若患者无法吞咽，应等待医疗人员处理。
- 不要让患者坐起或站立行走，这会导致血压下降；也不要忽视轻度症状，因为它们可能迅速恶化。
