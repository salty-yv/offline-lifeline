# 失温

## 先做这 3 步

1. **脱离寒冷并寻求遮蔽：**立即带伤者远离寒风、雨雪或冰冷水源，进入干燥温暖的帐篷、房屋或其他遮蔽处，阻断继续散热。
2. **脱去湿衣并保温核心：**小心脱掉湿衣物，用干燥衣物、毯子或急救毯包裹躯干和头部；可与同伴身体紧贴共享体温，并在腋下、胸部和腹股沟放置温暖的干布或热水袋（需用毛巾包裹）等物体。
3. **提供温暖且无咖啡因的饮品：**在伤者清醒且能吞咽时，少量多次饮用温开水、甜饮或含电解质的温热饮品，不要饮用酒精或咖啡因。

## 接下来几分钟

- **持续缓慢升温：**将伤者置于保温环境，避免骤热；可用自身体温或温暖的物品帮助缓慢升温。若手脚冰凉，可先温暖胸腹部而不是四肢，以防冷血回流引起心律失常或休克。
- **观察症状并求助：**失温患者可能出现发抖、言语含糊、神志不清或失去意识。若发现呼吸缓慢、心率异常或停止发抖，请立即呼叫急救服务，并参考 [休克](/medical/shock.md) 指南进行处理。
- **防止并发症：**当环境或衣物湿冷导致脱水，可适当参考 [脱水](/medical/dehydration.md) 指南补液；持续记录时间和采取的保温措施，为专业医疗人员提供信息。

## 不要做

- 不要给患者饮酒或咖啡，酒精和咖啡因会扩张血管或增加热量流失。
- 不要用明火、加热垫或过热的水直接接触皮肤，以免烫伤或引起心率波动；也不要给全身热水浴。
- 不要摩擦或揉搓身体，也不要让患者快速运动或按摩四肢，这些措施会将冷血流回心脏，导致低血压和心脏骤停。
