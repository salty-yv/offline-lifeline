# 失温

## 先做这 3 步

1. 离开风口、雨雪和湿冷地面。
2. 换下湿衣物，包裹头颈和躯干。
3. 清醒且能吞咽时少量补充温热饮品。

## 接下来几分钟

- 与同伴靠近保温。
- 意识模糊、停止发抖或行动笨拙时视为高风险。

## 不要做

- 不要喝酒保暖。
- 不要用过热物直接烫皮肤。
