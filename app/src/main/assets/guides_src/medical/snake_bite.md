# 蛇虫咬伤

## 先做这 3 步

1. 远离蛇虫，记住外观但不要追捕。
2. 减少活动，固定患肢。
3. 尽快联系救援或前往医疗点。

## 接下来几分钟

- 取下戒指、手表等可能勒紧的物品。
- 记录咬伤时间、症状变化。
- 呼吸困难、全身皮疹、面唇肿胀时按严重过敏处理。

## 不要做

- 不要吸毒液。
- 不要切开伤口或放血。
- 不要冰敷或饮酒。
