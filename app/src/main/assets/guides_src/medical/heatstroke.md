# 中暑

## 先做这 3 步

1. 停止活动，转移到阴凉通风处。
2. 解开过紧衣物，给颈部、腋下、腹股沟降温。
3. 清醒时小口补水和电解质。

## 接下来几分钟

- 观察是否意识混乱、抽搐、体温很高或无法喝水。
- 严重症状出现时尽快求救。

## 不要做

- 不要继续暴晒或赶路。
- 意识不清时不要强行喂水。
