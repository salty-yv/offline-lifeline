# 脱水

## 先做这 3 步

1. 停止不必要活动，转到阴凉处。
2. 少量多次喝水。
3. 有盐分或电解质时适量补充。

## 接下来几分钟

- 观察尿量、头晕、心慌和意识状态。
- 没有安全水源时，不要冒险饮用污染水。

## 不要做

- 不要一次喝下大量水。
- 不要喝酒或大量咖啡因饮料。
