# 脱水

## 先做这 3 步

1. **停止消耗并遮阴休息：** 一旦感觉口渴、头晕或尿量减少，立即停止不必要的体力活动，转移到阴凉处或通风良好的地方休息。
2. **小口补充液体：** 以少量多次的方式喝水或含电解质的饮品，如运动饮料或口服补液盐溶液，逐渐补充流失的水分和盐分。避免一次大量饮水，以免恶心或稀释血液中的电解质。
3. **需要时求医或使用口服补液：** 如果脱水伴随呕吐、腹泻或持续拒绝饮水，及时联系医生或使用药房购买的口服电解质溶液；婴幼儿应继续哺乳或配方奶，不宜仅用白水。

## 接下来几分钟

- 继续在阴凉处休息，观察尿量、心跳和意识状态；若出现皮肤苍白、脉搏加快或呼吸急促等严重脱水或休克征象，应参照 [休克](/medical/shock.md) 指南并尽快求救。
- 若身处炎热环境，避免再次暴晒；同时补充盐分可适当摄入含盐食物或口服补液盐；没有安全水源时，可以通过过滤、烧开或使用净水片等方式净化水源，不要直接饮用不明水源。
- 对于运动或高温导致的轻度脱水，可在体温恢复正常后继续少量进食和补水，并关注是否伴随中暑症状；参见 [中暑](/medical/heatstroke.md) 或 [热射病](/medical/heatstroke_severe.md) 指南。

## 不要做

- 不要一次性大量饮用水，暴饮可能导致恶心或稀释血液中的电解质。
- 不要饮用含咖啡因或酒精的饮料，它们会进一步加速脱水。
- 不要给婴幼儿饮用纯净水或运动饮料来代替口服补液溶液，也不要自行服用止泻药或止吐药，除非医生建议。
