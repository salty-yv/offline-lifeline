# 烧烫伤

## 先做这 3 步

1. 远离热源。
2. 用流动清洁冷水冲洗降温。
3. 用干净敷料或布料轻盖伤处。

## 接下来几分钟

- 取下附近戒指、手表等会勒紧的物品。
- 面部、手部、会阴、大面积或深度烧伤时尽快求救。

## 不要做

- 不要涂牙膏、油脂或不明药膏。
- 不要挑破水泡。
