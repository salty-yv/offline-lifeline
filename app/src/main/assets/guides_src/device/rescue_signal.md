# 求救信号

## 先做这 3 步

1. 准备一句固定求救信息：我在哪里、发生什么、几个人、伤情、电量。
2. 使用屏幕 SOS、闪光灯或地面标记制造重复信号。
3. 每隔一段时间发送或显示，避免持续耗电。

## 接下来几分钟

- 利用开阔处、反光物、鲜艳衣物提高可见性。
- 夜间优先使用屏幕高亮或间歇闪光。

## 不要做

- 不要长时间连续亮屏到耗尽电量。
- 不要离开已经留下信号的位置太远。
