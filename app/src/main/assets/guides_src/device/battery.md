# 电量保护

## 先做这 3 步

1. 降低屏幕亮度，关闭震动和不必要后台应用。
2. 关闭 Wi-Fi、蓝牙、定位或开启飞行模式。
3. 先查看离线指南，减少连续模型生成。

## 接下来几分钟

- 准备固定求救信息，间歇发送。
- 只在必要时打开屏幕 SOS 或闪光灯。

## 不要做

- 不要反复长时间对话。
- 不要持续录制视频或高亮照明。
