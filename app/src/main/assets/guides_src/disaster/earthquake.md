# 地震

## 先做这 3 步

1. 就近躲避，保护头颈。
2. 远离玻璃、悬挂物和高大家具。
3. 震动停止后再有序撤离。

## 接下来几分钟

- 注意余震和建筑物坠落风险。
- 受伤或被困时节省体力，用敲击或屏幕信号求救。

## 不要做

- 不要冲进受损建筑。
- 不要使用明火检查燃气泄漏。
