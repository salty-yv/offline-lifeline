# 泥石流

## 先做这 3 步

1. 发现河流突然断流或水流突然增大、听到类似轰鸣的巨响时，立即远离沟谷，沿两侧山坡向高处逃生。
2. 不要沿泥石流的流向奔跑，也不要爬树或停留在低洼地带，以免被卷入。
3. 到达安全区域后，用敲击、呼叫或亮光向外界发出求救，并拨打求救电话提醒救援人员和周围人员。

## 接下来几分钟

- 观察周围山体和沟谷，防止第二次泥石流或山体滑坡，不要立即返回危险区。
- 注意暴雨可能引发的洪水，避免涉水穿越被泥沙淤积的道路。
- 如有人受伤或被困，切勿单独进入滑坡区，应用手势或声音指引专业救援人员定位。

## 不要做

- 不要顺着泥石流方向逃跑或驻足围观。
- 不要涉水或穿越被泥石流覆盖的道路和桥梁。
- 不要为了取回物品或拍摄视频而返回危险区。
