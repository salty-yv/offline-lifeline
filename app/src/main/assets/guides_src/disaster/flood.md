# 洪水

## 先做这 3 步

1. 远离河道、低洼地和桥洞。
2. 能安全移动时向高处转移。
3. 保存电量，准备醒目的求救信号。

## 接下来几分钟

- 观察水位上涨速度和退路。
- 与同伴保持可见距离。

## 不要做

- 不要涉水强行通过。
- 不要在水边捡物品或拍摄。
