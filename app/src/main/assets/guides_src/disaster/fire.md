# 火灾

## 先做这 3 步

1. 判断烟和火的方向，朝安全出口或上风方向撤离。
2. 用湿布或衣物遮住口鼻，低姿移动。
3. 到安全区域后再呼救或发信号。

## 接下来几分钟

- 远离易燃物、山谷烟道和封闭空间。
- 被困时封堵门缝，靠窗或开阔处求救。

## 不要做

- 不要返回取物。
- 不要乘坐电梯。
