# 雷电

## 先做这 3 步

1. 离开山顶、空旷地、水边和孤立大树。
2. 分散同行人员，降低身体高度。
3. 关闭不必要电子设备，保存电量。

## 接下来几分钟

- 寻找低洼但不积水的位置。
- 等雷声远离后再移动。

## 不要做

- 不要躲在孤立大树下。
- 不要涉水或停留在金属护栏旁。
