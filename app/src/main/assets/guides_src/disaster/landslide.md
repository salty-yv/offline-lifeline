# 山体滑坡

## 先做这 3 步

1. 一旦收到滑坡预警或感到地面震动、听到山体塌落声，立即沿斜坡两侧或高处撤离，远离滑坡可能的路径。
2. 不要穿越有泥水或碎石的道路，选择安全路线迅速转移，并避免停留在河谷和低洼区域。
3. 在安全地点立即拨打紧急电话并提醒邻居，告知他们滑坡危险并帮助他们撤离。

## 接下来几分钟

- 留意降雨和山体状况，警惕可能的余滑或余震，不要靠近松动的岩石或树木。
- 滑坡发生后不要立刻返回现场，等待官方通知安全后再检查财物和损失。
- 如果有人员受伤或被困，在确保自身安全的前提下，使用手势或声音指引救援队伍的位置。

## 不要做

- 不要进入滑坡区域内部查看或尝试救人，次生滑坡随时可能发生。
- 不要停留在河谷或沟道内观看滑坡，滑坡和洪水可同时发生。
- 不要忽视官方发布的气象和地质灾害预警。
